#  NumPy Mastery: From Scratch to ML-Ready

###  Overview
This repository contains a deep-dive Jupyter Notebook designed to build a "Code from Scratch" foundation in NumPy. It covers everything from basic array properties to complex image processing and performance optimization techniques.

###  Key Modules
1.  **Foundations**: Creating 1D/2D arrays, understanding shape, dimension (`ndim`), and data types.
2.  **Indexing & Slicing**: Mastering "Data Ninja" skills to manipulate matrices and high-dimensional tensors.
3.  **Math & Broadcasting**: Element-wise operations and using the "Broadcasting Cheat Code" for incompatible shapes.
4.  **Aggregations**: Axis mastery (`axis=0` vs `axis=1`) using `.sum()`, `.mean()`, and `.argmax()`.
5.  **Masking & Fancy Indexing**: Logic-based data selection (e.g., selecting samples where `loss > threshold`).
6.  **Advanced Manipulation**: Reshaping, flattening, transposing, and expanding dimensions for ML compatibility.
7.  **Stacking & Splitting**: Combining features and splitting datasets into training batches.
8.  **Computer Vision Essentials**: Handling image batches `(N, H, W, C)`, normalization, and channel ordering.
9.  **Performance & Vectorization**: Writing fast, elegant code without `for-loops` to maximize efficiency.

###  Highlights
* **ML-Centric Exercises**: Practical challenges like implementing `Softmax`, feature-wise normalization, and mini-batch preparation.
* **Optimization Focused**: Comparisons of "Bad" vs. "Good" code to teach memory efficiency and vectorization.
* **3D/4D Tensor Support**: Specific guidance on handling image tensors and batch dimensions.

###  Prerequisites
* Python 3.x
* NumPy library (`pip install numpy`)
